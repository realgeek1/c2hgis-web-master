Introduction:

The Mapping Broadband Health in America tool is an interactive map created by the Connect2HealthFCC Task Force. It allows users to visualize, overlay and analyze broadband and health data at the national, state and county levels. By using the mapping tool as a starting point for looking at the broadband and health sectors, you can see the path to a more connected, healthier country. More.
